# Coding Arena Project Documentation

Welcome to the Coding Arena project documentation directory. This folder houses all architectural blueprints, design guidelines, and technical decisions.

## Folder Structure

- **`adr/`**: Architecture Decision Records (ADRs). Records design choices chronologically, using the format `NNNN-decision-title.md`.
- **`architecture/`**: System components, data models, webhooks, and communication specifications.

## What is an ADR?

An Architecture Decision Record (ADR) is a short text file capturing a significant design choice, the context, the trade-offs considered, and the final state. It ensures future contributors understand the _why_ behind core engineering layouts.
